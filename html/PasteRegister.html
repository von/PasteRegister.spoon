bindHotKeys
Signature PasteRegister:bindHotKeys(table)
Type Function
Description

Accepts a table of key bindings, e.g.:

{ chooser = {{"cmd", "alt"}, "c"}, load = {{"cmd", "alt"}, "l"}, paste = {{"cmd", "alt"}, "p"}, save = {{"cmd", "alt"}, "s"} }

Parameters:

  • table - Table of action to key mappings

Returns:
